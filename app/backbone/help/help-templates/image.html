Help with Images

Upload and annotate annotate an image

Image items allow you to upload image files and display them with zoom controls and the option to add annotations.

What to Know About Images

  • Supported file types: JPG, JPEG, PNG, and GIF
  • For animated GIFs, use the Google Drive and the Video content type
  • You can zoom in on a large image by using the zoom controls in the top left corner
  • You can add annotation text annotations to areas of an image
