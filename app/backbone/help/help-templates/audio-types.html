Help with Audio Types

Supported Media Types:

  • MP3: upload an mp3 audio file
  • Google Drive: A video hosted in Google Drive Instructions for Google Drive audio
  • BC Streaming Audio: audio hosted specifically on the Boston University streaming media server
  • Streaming Audio (RTMP):general format for audio hosted on a streaming server
  • URL: a link that takes users to audio on another website (not embedded in MediaKron)
