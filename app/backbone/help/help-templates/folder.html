Help with Folders

Group related content with a table of contents

Folders are one of the basic building blocks of MediaKron site. They help organize a site by allowing you to group related content and put items into a particular order.

What to Know About Folders

  • Any MediaKron content can be put into a folder
  • The contents of a folder can be put an any order you want
  • You can navigate through a folder using next/previous navigation on the left and right sides of the screen. 
  • Folders can be nested in other folders and breadcrumb trail shows where you are in the folder hierarchy. 

When to Use a Folder 

  • Create major sections of a site by creating folders and making them menu items
  • Use sub-folders to organize content hierarchically.
  • Create areas of the site in which groups can collaborate
