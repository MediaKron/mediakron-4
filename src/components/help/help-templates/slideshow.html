Help with Slideshows

Show items in a series of slides

A slideshow is a collection in which content is viewed one slide at a time in a sequence.

All content types can be added to a slideshow except other slideshows.

Example Slideshow

Layout Options

  • Grid: A grid layout with square thumbnails. If you've uploaded an image for the slideshow, it will appear as a top banner.
  • Mosaic: A grid layout with images in their original aspect ratios
  • Full Page: A full-page background image if present.
